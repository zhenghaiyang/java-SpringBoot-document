Java下载

去java官网下载JDK1.8 因为目前市面上1.8用的比较多，你也可以下载新版本

官网下载连接

配置JAVA环境变量

1).打开我的电脑,右键属性

选择【系统变量】点击【新建】

变量名--- JAVA_HOME

变量值--- 安装JDK的位置

点击 【确定】

继续点击 【新建】

变量名--- CLASSPATH

变量值--- .;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar 【注意前面那个 . 】

点击 【确定】

选择 【系统变量】里的“path”变量 新建变量--- %JAVA_HOME%\bin (可能大家打开不一样)

打开控制台 输入 【java -version】回车 会显示JAVA版本表示成功

Idea下载

去idea官网下载 --选择社区版，你也可以去别的地方下载(idea有收费和免费，我们下载社区版免费的，基本上已经满足开发需求)

安装IDEA

一路next

创建桌面应用选择 64-bit- launcher

选择.java .groovy .kt ,增加环境变量

继续next

选择安装路径

点击【instal】

instal 中请等待

华丽分割线

点击【finish】

我们是第一次使用，直接选择 Do Not import settings

华丽分割线

选择我同意条款 点击【continue】

到这里idea 已经彻底安装完成啦。下面是初始化一些配置项,我们选择跳过-选择默认设置(skip remaining and set defaults)

打开Idea,这里只介绍Idea的安装(如何使用Idea创建一个 Java Spring Boot项目请查看别的章节)

Maven下载

去Mava官网下载,这里有很多中下载,我们选择下载压缩包

官网下载连接

下载后将它解压, 我们会得到一个解压包,这就是maven的东西。(把这个包放到一个地方，以后我们会用得着)

点开进去，conf文件夹里面有一个 settings.xml ，我们打开他

我们搜索 mirrors 关键字 ，我们把阿里的maven下载地址放上去,方便我们下载一些依赖包

                  <mirror>  
                    <id>nexus-aliyun</id>  
                    <mirrorOf>central</mirrorOf>    
                    <name>Nexus aliyun</name>  
                    <url>http://maven.aliyun.com/nexus/content/groups/public</url>  
                  </mirror>
